## 🛠️ Usage

### 💻 Installation

Clone the repository.

```bash
git clone git@github.com:ragibasif/dlog.git
```

Make sure you have `gcc` and `make` installed.

### 🔧 Build the project

```bash
make
```

### 🧹 Clean build files

```bash
make clean
```

### 🏃 Run the executable

```bash
./dlog
```
